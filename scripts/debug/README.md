# Debug Scripts

This directory contains temporary debugging and testing scripts.

## Scripts

- `check_clickhouse.ts` - Debug script for ClickHouse data inspection
- `check_weight_flow.ts` - Debug script for weight data flow
- `check_drivers.ts` - Debug script for forecast drivers
- `check-data-range.ts` - Debug script for data ranges
- `test-correlation-engine.ts` - Test script for correlation engine

## Usage

These scripts are for debugging purposes and may contain hardcoded user IDs. 
Review and modify as needed before running.
